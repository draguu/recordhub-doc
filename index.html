Viewport Setup

Create Recording Viewport

The addon records from a separate viewport that follows your main workspace.

Quick Setup (Recommended)

  1. Click "Create Recording Viewport" button in panel
  2. A small viewport appears at bottom (20% height)
  3. This viewport auto-copies your main view
  4. Keep this window visible during recording
üí° Pro Tip: The recording viewport has overlays disabled for cleaner output

Manual Setup

  1. Split your viewport: Right-click edge ‚Üí Split Area
  2. Drag to create second viewport at bottom/side
  3. Position both viewports side-by-side or stacked
  4. The smaller viewport will be used for recording

Frame Guide Overlay

Toggle frame guide to preview exact recording area:

  • Click eye icon to enable/disable overlay
  • Shows resolution boundaries in viewport
  • Adjust opacity and color in panel
  • Helps frame your shots perfectly

Cleanup

After recording, remove extra viewport:

  • Click "Remove Extra Viewports" button
  • Joins viewports back to single window
  • Or manually: Join Area from viewport edge menu
‚ö†Ô∏è Important: Keep the recording viewport visible throughout your session. Minimizing it will stop capture.

Operating Modes

OrbitContinuous 360¬∞

Smooth rotation around Z-axis. Perfect for showcasing models.

  • Speed: 0.1-90¬∞/sec
  • Smoothness: Camera interpolation
  • Track selection or entire objects
  • Auto-zoom: 10-300%

Ping-PongBack & Forth

Rotates within defined range. Great for reveals.

  • Range: 10-360¬∞
  • 5 easing modes (Linear, Ease In/Out, Sine)
  • Smooth direction transitions
  • Speed and smoothness controls

Follow MeViewport Copy

Records your viewport movements with delay.

  • Hold time: 0.1-5 sec (wait before recording angle)
  • Travel time: 0.3-10 sec (smooth transition speed)
  • Needs 2+ viewport windows
  • Works in any editing mode

Quick Guide

Recording Workflow

  1. Create recording viewport: Click button or split manually
  2. Select mode: Orbit / Ping-Pong / Follow Me
  3. Adjust settings: Speed, smoothness, tracking
  4. Configure output: Resolution, video mode, FPS
  5. Preview: Test before recording
  6. Record: Press ESC to stop
  7. Cleanup: Remove extra viewports when done

Key Settings

Speed: Rotation speed (degrees/sec)

Smoothness: Camera interpolation (0.01-1.0)

Track Selection: Follow selected objects/faces/edges

Auto Zoom: Adjust distance automatically

Resolution Settings

Choose output resolution in Settings:

  • Viewport: Uses current viewport size
  • Active Camera: Matches camera resolution
  • Presets: HD 720p, Full HD 1080p, 2K, 4K, Reel (1080x1920)
  • Custom: Set specific width √ó height
üí° Frame Guide: Enable overlay to see exact recording area in viewport

Export Options

Click Settings button:

  • Mode: Timelapse (sped up) or Realtime
  • FPS: Output frame rate (24-120)
  • Speed: Timelapse multiplier (2-60x)
  • Quality: Fast / Medium / High
  • Cleanup: Auto-delete frames after export
üí° Best Settings: Speed 2-5¬∞/sec, Smoothness 0.2-0.4, Quality Medium, 1080p

Output Files

Saved as: recording_YYYYMMDD_HHMMSS_[mode].mp4

Default location: ~/Timelapse_Recordings

Troubleshooting

FFmpeg Not Found

  • Install from ffmpeg.org
  • Add to PATH or set in Settings
  • Test: ffmpeg -version

No Frames Captured

  • Ensure recording viewport is visible
  • Check write permissions to output folder
  • Verify disk space available
  • Recording viewport must not be minimized

Choppy Video

  • Increase smoothness (0.3-0.5)
  • Lower rotation speed
  • Use higher FPS output
  • Reduce viewport shading complexity

Follow Me Not Working

  • Need 2+ viewport windows open
  • Hold view still for full delay time
  • Recording window must be separate from work window
  • Don't move view during hold time

Wrong Resolution Output

  • Enable frame guide to preview recording area
  • Check resolution mode setting
  • For custom sizes, verify width/height values
  • Viewport mode uses actual viewport size

Performance Issues

  • Use solid shading instead of rendered
  • Close unused applications
  • Record at lower resolutions first
  • Reduce viewport size
  • Disable overlays in recording viewport

Can't Remove Recording Viewport

  • Use "Remove Extra Viewports" button
  • Or manually: Right-click viewport edge ‚Üí Join Area
  • Drag cursor into viewport to keep
‚ö†Ô∏è Still stuck? Check Blender console (Window ‚Üí Toggle System Console) for errors.

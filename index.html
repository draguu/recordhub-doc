Quick Guide

Recording

  1. Select mode: Orbit / Ping-Pong / Follow Me
  2. Adjust settings: Speed, smoothness, tracking
  3. Preview: Test before recording
  4. Record: Press ESC to stop

Key Settings

Speed: Rotation speed (degrees/sec)

Smoothness: Camera interpolation (0.01-1.0)

Track Selection: Follow selected objects/faces

Auto Zoom: Adjust distance automatically

Export Options

Click Settings button:

  • Mode: Timelapse (sped up) or Realtime
  • FPS: Output frame rate (24-120)
  • Speed: Timelapse multiplier (2-60x)
  • Quality: Fast / Medium / High
  • Cleanup: Auto-delete frames after export
üí° Best Settings: Speed 2-5¬∞/sec, Smoothness 0.2-0.4, Quality Medium

Output Files

Saved as: recording_YYYYMMDD_HHMMSS_[mode].mp4

Default location: ~/Timelapse_Recordings

Troubleshooting

FFmpeg Not Found

  • Install from ffmpeg.org
  • Add to PATH or set in Settings
  • Test: ffmpeg -version

No Frames Captured

  • Check 3D viewport is active
  • Verify write permissions
  • Check disk space

Choppy Video

  • Increase smoothness (0.3-0.5)
  • Lower rotation speed
  • Use higher FPS

Follow Me Not Working

  • Need 2+ viewport windows
  • Hold view for full delay time
  • Recording window must be separate

Performance Issues

  • Use solid shading, not rendered
  • Close unused apps
  • Record shorter clips
  • Reduce viewport size
